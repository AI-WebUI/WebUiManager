PS：div和行内元素改变display属性为inline-block的时候会有间隙问题(实例)

解决方案：


    <!--设置父元素的font-size为0可以解除间隙-->
    <!--注意在ie浏览器的兼容模式下div的display：inline-block是不起作用-->
    <!--想让块级元素在一行内显示改变span的display属性为inline-block或者用浮动布局-->
    .container{
    text-align: center;
    font-size:0;
    }
    .inline-block{
    display: inline-block;
    width:200px;
    height:100px;
    border:1px solid red;
    }
    <div class="container">
        <div class="inline-block"></div>
        <div class="inline-block"></div>
        <div class="inline-block"></div>
    </div>

三、如何让一个块级元素一直在屏幕中心不随页面滚动而位置变动（兼容性高模式）

html跟css代码实例


    <!--文档最外层是html跟body让固定的元素跟body脱离关系设置overflow：hidden-->
    html,body {
    overflow:hidden;
    margin:0px;
    width:100%;
    height:100%;
    }
    <!--第二部假造一个body，让它的内容溢出可以显示-->
    .virtual_body {
    width:100%;
    height:100%;
    overflow-y:scroll;
    overflow-x:auto;
    }
    .container{
    height:888px;
    }
    <!--固定你想要元素让它显示在屏幕中心这样不管假body怎么动它位置都不会变化了-->
    .fixed_div {
    position:absolute;
    z-index:2008;
    top:50%;
    left:50%;
    width:200px;
    height:200px;
    border:1px solid red;
    background:#e5e5e5;
    margin-top:-100px;
    margin-left:-100px;
    }
    <div class="fixed_div">CENTER</div>
        <div class="virtual_body">
            <div class="container">
            I am content !
            </div>
         </div>
